# House Price Exploratory Data Analysis (EDA)

## Overview

This project presents an Exploratory Data Analysis (EDA) of the Boston Housing dataset using Python.

The objective is to understand the dataset, identify meaningful patterns, analyze relationships between variables, handle missing values, and prepare the data for future machine learning tasks.

---

## Objectives

- Explore the structure of the dataset
- Perform descriptive statistical analysis
- Analyze feature distributions
- Detect and handle missing values
- Examine feature correlations
- Visualize important relationships
- Draw insights from the data

---

## Technologies Used

- Python
- Pandas

- Seaborn
- Jupyter Notebook

## Key Findings

Some variables show a strong relationship with house prices, particularly:

- Average number of rooms (RM)
- Percentage of lower status population (LSTAT)

Correlation analysis and visualizations provide useful insights into the factors influencing housing prices.
